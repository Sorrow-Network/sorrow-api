# 📦 SorrowAPI
This might just be the sussiest little api out there.

## 📝 Usage
```js
const sorrowapi = require('sorrow-api'); 
sorrowapi.joke()
    .then(url => console.log(url))
    .catch((error) => console.error(error));
```
